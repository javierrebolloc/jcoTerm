# Session 2026-06-24 — Review Fixes + UX Improvements

## Summary

Code review cycle (2 rounds), bug fixes from user testing, and 15 UX improvements. Terminal stability fixes, SFTP enhancements, custom menu bar, session search, and keyboard shortcuts.

## Bugs Fixed

### xterm.js Race Conditions
- **Disposed Viewport error** — React Strict Mode causes mount→cleanup→mount; xterm's `Viewport._innerRefresh` fires via pending `requestAnimationFrame` after dispose. Fixed with zero-size guards on all `fitAddon.fit()` call sites (openTerminal, ResizeObserver, isActive effect).
- **SSH output before term.open()** — Output arriving before terminal opens caused `dimensions` error. Fixed by buffering writes in `pendingWritesRef` and flushing after successful open.
- **Terminal corruption on SFTP↔SSH switch** — `display:none` destroyed container dimensions; xterm calculated wrong cols/rows on return. Fixed by stacking panels with `position:absolute` + `visibility:hidden` which preserves layout.
- **Redundant ssh.resize on view switch** — `fitAndResize()` sent resize even when cols/rows hadn't changed, causing server to redraw prompt. Fixed with `lastSizeRef` dedup.

### SFTP
- **Upload EBUSY leaves 0-byte file** — `readStream.on('error')` now destroys writeStream and unlinks the remote file.
- **editRemote: saves during upload lost** — `pendingReupload` flag re-triggers upload after current one finishes.
- **editRemote: no cleanup on quit** — `activeEditCleanups` Set tracks watchers; `cleanupEditWatchers()` called from `before-quit`.
- **editRemote: 1-hour timer never cancelled** — `maxTimer` stored and cleared in `cleanup()`.
- **editRemote: raw error message** — Now sanitized like `withSftpSession` (permission denied, not found, generic).
- **editRemote: Windows-invalid chars** — `sanitizeFileName()` replaces `<>:"|?*` and control chars.
- **editRemote: missing file guard** — `fs.existsSync` check before upload; calls `cleanup()` if file deleted.
- **Column resize re-renders per pixel** — Now uses `th.style.width` during drag, `setColWidths` only on pointerUp.
- **sorted array reference instability** — `useMemo` for sorted entries prevents useEffect keydown re-subscription.
- **Lost pointer capture** — `onLostPointerCapture` handler resets resize refs.
- **Sort triggered during resize** — `didResizeRef` flag blocks `handleSort` after resize.

### Sessions
- **SaveSessionModal loses sortOrder** — Added `sortOrder: initialSession?.sortOrder` to preserve position on edit.
- **Right-click doesn't select** — `handleContextMenu` now selects the clicked file before showing menu.

### General
- **Title "SSH AI Client"** — Changed to "jcoTerm" in index.html.
- **Language change not working** — `handleSave` now calls `onClose()` which triggers `refreshProviderStatus` → `setLanguage`.
- **TerminalPane disconnect inconsistency** — Now uses `onClose(sshSessionId)` instead of direct `ssh.disconnect` IPC.

## New Features

### Custom Menu Bar
- jcoTerm menu: Settings (Ctrl+,), Quit (Ctrl+Q)
- Edit: standard clipboard operations
- View: reload, devtools, zoom, fullscreen
- Help: About (opens Settings → About section)
- IPC channels: `app:menuOpenSettings`, `app:menuOpenAbout`

### SFTP Enhancements
- **Edit remote files** — Right-click → Edit: downloads to temp, opens in default editor, auto re-uploads on save.
- **Edit local files** — Same right-click → Edit option, opens with `shell.openPath`.
- **Resizable columns** — Drag column borders in file table header.
- **Local mkdir/delete** — Functional buttons (were no-ops). IPC `local:mkdir`, `local:delete` with home-dir validation.
- **Delete key** — Pressing Delete on selected files triggers delete with confirmation.
- **F5/F7 shortcuts** — F5=Refresh, F7=New folder in SFTP panes.
- **Transfer error toast** — Failed transfers show error toast in the UI.

### Terminal
- **Flexible splits** — 2/4/8 splits allowed with fewer sessions; empty cells show placeholder, auto-fill on connect.
- **Middle-click close tab** — Standard browser pattern.
- **Ctrl+1..9** — Jump to tab by position.
- **ARIA roles** — `role="tablist"`, `role="tab"`, `aria-selected` on tab bar.

### AI Chat
- **Copy button** — Per-message copy button (visible on hover).
- **Clear conversation** — Button in tab bar to reset active chat.
- **Ctrl+Shift+A** — Toggle AI panel visibility.

### Sessions
- **Search/filter** — Input in sidebar filters by name, host, or username.
- **Session reorder** — Drag & drop between sessions to reorder (sortOrder field). Visual before/after indicators.

### Keyboard Shortcuts
- **Ctrl+Shift+S** — Toggle SSH↔SFTP view.
- **Ctrl+Shift+A** — Toggle AI panel.

- **F5** — Refresh (SFTP).
