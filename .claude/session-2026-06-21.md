### 2. Settings Redesign with Sidebar

- 700px modal with left sidebar and 5 sections: General, Terminal, AI, Connections, About
- New settings: fontFamily, cursorStyle, cursorBlink
- Settings applied live to terminal via `term.options.*`
- Functional language selector
- About: author, email, MIT license, version

### 3. SFTP Manager (FileZilla-like)

**Backend:**
- 8 new SFTP methods in SshSession: stat, mkdir, rmdir, unlink, rename, chmod, download (streaming), upload (streaming)
- `formatPermissions()` — Unix mode to string
- local.handlers.ts: listDir, homePath, drives (Windows A:-Z:)

**Frontend (16 components):**
- SftpManager — container with multi-tab remote, error toast
- SftpConnectionBar — session selector + connect/disconnect
- SftpFilePane — reusable panel (local/remote) with toolbar
- SftpFileTable — table with multi-select (Ctrl/Shift), sorting, keyboard search, drag & drop
- SftpBreadcrumb — path navigation with auto-scroll
- SftpContextMenu — context menu
- SftpTransferQueue — queue with progress and errors
- SftpChmodDialog — 3x3 editor + octal

**Features:**
- Dual-pane: local (left) + remote (right)
- Multiple remote connections with tabs
- Upload/download with real-time progress
- Drag & drop between panels (multi-file)
- Multi-selection (Ctrl+click, Shift+click)
- Keyboard search (type letters to search)
- mkdir, delete, rename, chmod
- Credential dialog if the session has no password
- Transfer errors shown in queue and toast

### 4. i18n System

- Custom engine: `t(key, params?)`, `setLocale()`, `getLocale()`
- Flat JSON with dot-separated keys (~250 keys)
- `en.json` (English, default) + `es.json` (Spanish)
- React: LanguageProvider + useTranslation hook
- Main process: direct import of `t()`
- ~20 renderer components updated
- ~12 main handlers updated
- Restart notice when changing language

### 5. Multi-exec in Split

- "Multi-exec" option in Split menu
- Input replicated to all split terminals
- Each terminal has ">>" toggle to exclude itself
- Automatically disabled when exiting split

### 6. Custom Confirmation Dialogs

- ConfirmProvider + useConfirm() replace `window.confirm()`
- Dark theme, red button for destructive actions
- 5 usage points: delete session/folder/credential/SFTP file, accept host key

### 7. Bugfixes

- Black screen: `handleConnect` referenced before initialization (temporal dead zone)
- xterm.js dimensions: try-catch with retry on `term.open()`
- Upload 0%: handler returned transferId AFTER the stream (now in background)
- Chmod input: non-octal character filter, no sync loop
- ConnectDialog: misaligned CSS classes (.dialog vs .modal)
- Modals don't close on outside click (intentional, per user request)
- autoFocus on username field (not on password)
- Removed key emoji from Credentials, removed "+" from New session
- Autofill.enable error suppressed (removed DevTools auto-open)
- Multi-exec paste: Shift+right-click pastes in all split terminals
- Terminal cut off in split: reduced padding, compact statusBar, min-width/min-height on grid cells
